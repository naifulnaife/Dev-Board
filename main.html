Question-1: What are the different ways to select an element in the DOM?

1.getElementsByTagName("tag") 2.getElementById("id") 3.getElementsByClass("class") 4.querySelectorAll("selector") 5.querySelector("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets or gets HTML content(including tags),innerText gets only visible text(ignoring hidden elements,textContent gets all text(including hidden))

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a propagation mechanism where an event starts from the target and by time it moves to its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

By using document.createElement("tag") to create an element,appendChild() to add and removeChild() to remove from the DOM.
